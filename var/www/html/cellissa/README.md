\# Sistema de Gestión - Cellissa Comunicaciones



Proyecto desarrollado como actividad final de la asignatura \*\*Desarrollo de Proyectos con Software Libre\*\*.



\## Descripción

Este sistema web fue diseñado para gestionar procesos básicos de un taller de electrónicos, permitiendo administrar clientes, equipos, reparaciones y pagos.



\## Tecnologías utilizadas

\- Ubuntu Server / Ubuntu en máquina virtual

\- Apache

\- PHP

\- MySQL

\- HTML / CSS



\## Módulos implementados

\- Gestión de Clientes

\- Gestión de Equipos

\- Gestión de Reparaciones

\- Gestión de Pagos



\## Estructura del proyecto

\- `index.php`: página principal

\- `conexion.php`: conexión con la base de datos

\- `clientes.php`: módulo de clientes

\- `equipos.php`: módulo de equipos

\- `reparaciones.php`: módulo de reparaciones

\- `pagos.php`: módulo de pagos

\- `style.css`: estilos del sistema

\- `database.sql`: script de creación de base de datos



\## Base de datos

La base de datos utilizada es `taller\_electronicos`.



\## Autoría

Proyecto académico realizado para fines educativos.
